MyProject: Zebrafish Scale Detection and Illumination Optimization with Retina Net

Christian Richardson

cr304@duke.edu

Zebrafish scale segmentation is a problem that has yet to be solved by known image processing approaches. Zebrafish scales are relatively thick three-dimensional structures that must be imaged in 2D slices which are compiled into 3-dimensional tiff stacks. Segmenting images of these scales is difficult due to the low contrast of the images in areas with poor illumination, and the close proximity that scales share, which generally leads to multiple scales being merged into a single connected component when image segmentation is used. I propose an object detection-based solution, where the object detection model Retina Net is used to regress bounding boxes around individual scales. Retina Net is modified to include a multiply layer that element-wise multiples the sample intensity map, or the image of a scale, with a learned illumination map. Unfortunately, I was not able to implement a functional Retina Net model that includes a custom illumination layer. Instead, the classic Retina Net model was used to perform object detection on Zebrafish scale images.

Retina Net Implementations Used: https://github.com/fizyr/keras-retinanet. https://keras.io/examples/vision/retinanet/.
